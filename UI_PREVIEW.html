ScreenSlate - UI Layout

This is a preview of the ScreenSlate application UI with 4 main regions:

  • Sources List (Left): Shows available recording sources with empty state
  • Preview/Status (Center): Displays recording preview or idle status
  • Settings Panel (Right): Configure FPS (default 30), output directory (Downloads), and microphone toggle
  • Logs Panel (Bottom): Timestamped logs with levels, clear button, and auto-scroll
